# Site Map: Pages and Components

*Rebuilt 2026-07-13 ΓÇË the previous version of this file (dated 2024-03-19) referenced
components that no longer exist (`Countdown.svelte`, `BookCallout.svelte`,
`FuturePredictions.svelte`, `DiscordButton.svelte`) and is not a reliable guide to the
current site. This version is generated by reading the actual route tree and import
statements, not carried forward by hand.*

Every page and every component gets a stable ID (`PG-NN` / `CM-NN`) so a bug report,
QA note, or commit message can reference "PG-17" instead of a full path. IDs are
assigned by directory order and are not meant to imply priority.
